
Hub删除(1)删除Github上现有的存储库.= =

## SYNOPSIS

"hub删除"[Y][<organization>/]<NAME>

## OPTIONS

-   y,是:跳过确认提示并立即删除存储库.

-   [<ORGANIZATION>/]<NAME>Github上存储库的名称.

## EXAMPLES

```
$ hub delete recipes
[ personal repo deleted on GitHub ]

$ hub delete sinatra/recipes
[ repo deleted in GitHub organization ]
```
