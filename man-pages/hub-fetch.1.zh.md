
hub获取(1)在执行Git之前添加丢失的远程文件

# fetch.

## SYNOPSIS

"hub读取"<USER>[,<USER2>...…]

## EXAMPLES

```
$ hub fetch ‐‐multiple jingweno mislav
> git remote add jingweno git://github.com/jingweno/REPO.git
> git remote add mislav git://github.com/mislav/REPO.git
> git fetch jingweno
> git fetch mislav
```
